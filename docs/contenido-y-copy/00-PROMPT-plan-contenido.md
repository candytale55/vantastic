Para planificar **el contenido de una web (copy + estructura narrativa) a partir de un arquetipo de marca y 3 buyer personas**, conviene seguir un proceso ordenado. Esto evita que el contenido quede genérico y hace que cada sección de la web responda a una intención del usuario.

Te dejo un **proceso profesional simplificado** (muy usado en marketing y UX copywriting) y después **prompts útiles** para trabajar con IA.

---

# 1. Definir la estrategia de contenido

Antes de escribir, hay que convertir el arquetipo y los buyers en **decisiones de comunicación**.

### 1.1 Identidad de marca

Extraer de tu arquetipo:

* personalidad de la marca
* tono de voz
* emociones que debe provocar
* valores

Ejemplo:

| Elemento  | Resultado           |
| --------- | ------------------- |
| Arquetipo | Explorador          |
| Emoción   | libertad, aventura  |
| Tono      | cercano, inspirador |
| Promesa   | viajar a tu ritmo   |

---

### 1.2 Mapear los 3 buyer personas

Para cada uno define:

* motivación principal
* miedo o fricción
* objetivo del viaje
* qué valoran en una van vintage

Ejemplo:

| Buyer    | Motivación         | Fricción    | Qué busca            |
| -------- | ------------------ | ----------- | -------------------- |
| Parejas  | escapada romántica | incomodidad | experiencia especial |
| Viajeros | aventura           | precio      | flexibilidad         |
| Turistas | experiencia local  | logística   | facilidad            |

Esto **determina el contenido de cada sección**.

---

# 2. Mapear el contenido de la web

Luego vinculas **secciones → objetivo → buyer persona**.

Ejemplo típico para este negocio:

| Sección       | Objetivo           |
| ------------- | ------------------ |
| Hero          | transmitir emoción |
| Cómo funciona | reducir fricción   |
| Flota de vans | deseo              |
| Experiencias  | storytelling       |
| Testimonios   | confianza          |
| FAQ           | objeciones         |
| CTA           | conversión         |

---

# 3. Crear el “mensaje principal”

Toda la web debe responder a:

**¿Por qué alquilar una van vintage y no otra cosa?**

Fórmula útil:

```
Experiencia + Beneficio emocional + Diferenciador
```

Ejemplo:

> Viaja por la costa en una furgoneta vintage y convierte tu escapada en una aventura inolvidable.

---

# 4. Escribir cada sección con una intención

Ejemplo:

### Hero

Debe responder en 3 segundos:

* qué es
* para quién
* qué emoción produce

Ejemplo:

**Título**

> Escapa de la rutina en una van vintage.

**Subtítulo**

> Alquila una furgoneta clásica y explora la costa a tu ritmo.

---

### Sección "Cómo funciona"

Objetivo: eliminar miedo.

```
1. Elige tu van
2. Reserva online
3. Recoge y viaja
```

---

### Flota

Objetivo: deseo + diferenciación

Cada van debe tener:

* personalidad
* historia
* estilo de viaje

---

# 5. Ajustar SEO

Antes de escribir conviene definir:

* keyword principal
* keywords secundarias

Ejemplo:

* alquiler furgoneta camper vintage
* alquiler van clásica
* camper retro España

---

# 6. Crear un calendario de contenidos

Esto alimenta SEO y redes.

Ejemplos de artículos:

* rutas en van por Valencia
* cómo viajar en camper vintage
* escapadas románticas en furgoneta

# PROMPTS ÚTILES

Aquí tienes prompts muy eficaces.

---

# 1. Analizar arquetipo + buyers

Prompt:

```
Actúa como estratega de marca y UX writer.

Tengo una marca de alquiler de vans vintage.

Arquetipo de marca:
[pegar arquetipo]

Buyer personas:
[pegar los 3]

Analiza:

1. Qué emociones debe transmitir la marca
2. Qué mensaje principal debería comunicar
3. Qué tipo de tono de voz usar
4. Qué promesa de valor debe destacar

Resume el resultado en una guía de comunicación para la web.
```

---

# 2. Generar estructura narrativa de la web

```
Actúa como estratega de contenido y copywriter.

Estoy creando una web de alquiler de vans vintage.

Arquetipo de marca:
[pegar]

Buyer personas:
[pegar]

Propón una estructura completa de la web con secciones optimizadas para conversión.

Para cada sección explica:
- objetivo
- mensaje principal
- tipo de contenido
```

---

# 3. Escribir el copy de la home

```
Actúa como copywriter especializado en turismo experiencial.

Escribe el contenido de la página principal para una empresa de alquiler de vans vintage.

Incluye:
- Hero
- Beneficios
- Cómo funciona
- Flota
- Experiencias
- Testimonios
- CTA

Tono:
aventura, libertad, nostalgia.
```

---

# 4. Adaptar contenido a cada buyer persona

```
Analiza estos tres buyer personas y reescribe los beneficios del servicio para que conecten con cada uno.

Buyer personas:
[pegar]

Resultado:
3 versiones de copy adaptadas.
```

---

# 5. Crear storytelling para las vans

Esto funciona muy bien.

```
Escribe descripciones emocionales para cada una de estas vans vintage.

Cada van debe tener:
- nombre
- personalidad
- tipo de viaje ideal
- experiencia que ofrece

Vans:
[lista]
```

---

# 6. Crear FAQ que conviertan

```
Genera una sección FAQ para una empresa de alquiler de vans vintage.

Incluye preguntas reales que tengan los clientes sobre:

- seguro
- conducción
- rutas
- equipamiento
- reservas

Responde de forma clara y amigable.
```

---

# 7. Crear contenido SEO

```
Genera 15 ideas de artículos para un blog sobre viajes en vans vintage en España.

Cada idea debe incluir:
- título SEO
- keyword principal
- intención de búsqueda
```
